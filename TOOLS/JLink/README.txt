
Requirements
============

libusb 1.0
============
libusb 1.0 is used by the J-Link software to communicate with J-Link debug probe via USB.
Please make sure that this library is installed on your system. 
You can install it by running the following command in a terminal window:

sudo port install libusb

This command will download and install libusb on your computer. 
Please unplug and replug J-Link after successful installation.
